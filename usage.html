Welcome! A quick rundown of how this application works, before you return.

First off. It appears to only work on Microsoft Edge unfortunately.
Chrome was having massive challenges with the multi-field values. So when using
voice capture to record the nth element, all previously recorded elements were
overwritten with the new data. Webstorm suggested edge and it worked so, here we are.

Each text-field will activate recording upon clicking into it.
Upon clicking out of a field, or to another field, that field stops recording.

Upon submit, the application will read-back what it captured, or if you did
not speak but choose to put in text, then the text will be spoken back.

That's it. Simple as can be. The info button, if you haven't clicked it, speaks
some information such as my name, and some class info.
